# 🚀 Velora Trading Engine

AI-powered algorithmic trading platform for Forex & Crypto — built for automation, precision, and scale.

---

## ⚡ Overview

Velora is a full-stack trading system that combines:

* 📊 Market data analysis
* 🤖 AI-driven decision engine
* ⚙️ Automated MT5 execution
* 📈 Performance tracking

---

## 🧠 Features

* Automated trading (MT5)
* Strategy engine (EMA + RSI)
* Risk management (1% rule)
* Web + Mobile dashboard

---

## 🏗️ Architecture

/backend → FastAPI backend
/web_frontend → Web app
/velora_flutter → Mobile app

---

## 🚀 Setup

```bash
git clone https://github.com/Rishi2119/velora-trading-engine.git
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
