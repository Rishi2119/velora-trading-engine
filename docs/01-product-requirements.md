# 01 - Product Requirements (PRD)

## Overview
Velora Autonomous Trading Agent is a robust, persistent AI worker designed for highly automated trading via autonomous logic and strict system instructions.

## Goals
- Automate MT5 trading using the Kimi K2.5 LLM.
- Provide real-time transparency of the AI's thoughts to the Velora app.
- Ensure strict risk control and avoid catastrophic failures.

## Scope
- **In Scope**: Python continuous loop, LLM reasoning, MT5 local execution, logical JSON memory, mobile status API interfaces.
- **Out of Scope**: High-frequency trading (HFT), multi-tenant cloud distributions.
