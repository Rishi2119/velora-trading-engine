# Product Requirements Document

## Vision
To build a fully autonomous AI agent capable of logical reasoning and independent MT5 trade execution.

## Features
1. **24/7 Loop**: Unattended background processing.
2. **Reasoning Engine**: LLM-driven deterministic decisions focusing on strict rules.
3. **Memory Storage**: Persistence of state and previous LLM contexts.
4. **Mobile Control**: Start/stop/monitor commands manageable from the Velora app.
5. **Safety Guardrails**: Strict execution boundaries and maximum retry limits.
