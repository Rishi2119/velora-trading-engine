# Web App (Next.js 14)

## Stack
- Next.js App Router
- TailwindCSS
- Supabase SSR Auth
- Zustand + React Query (ready)

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Protected Routes
- `/dashboard`
- `/profile`

Protected by `middleware.ts` using Supabase session cookies.
